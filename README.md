# Personal_Blog-Django
- This is a personal blog made using python's web framework django. It comes with feature such as word counter. 
- It follows public blog system. 
- It uses django's inbuilt SQLITE database.
- Tools Used:
  1. Python 
  2. Django
  3. HTML5
  4. CSS3
  5. Bootstrap 4.0
  6. SQLITE3
